这是一篇示例文章，展示如何在 Obsidian 中写作并发布到 GitHub Pages。

## 如何发布文章

在文章开头添加 frontmatter：

```yaml
---
title: "文章标题"
date: 2026-01-16
tags: [标签1, 标签2]
published: true  # 设置为 true 才会发布
---
```

## 支持的功能

- **Markdown 语法**：完整支持标准 Markdown
- **代码高亮**：使用三个反引号包裹代码
- **标签分类**：可以给文章添加多个标签
- **深浅模式**：点击右上角按钮切换主题

## 写作流程

1. 在 Obsidian 的 `03-Writing/01-Tech-Blog/` 文件夹中写作
2. 完成后设置 `published: true`
3. 将文件复制到 GitHub repo 的 `_posts` 文件夹
4. Push 到 GitHub，自动部署

就这么简单！
